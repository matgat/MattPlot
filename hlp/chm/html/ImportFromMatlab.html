Importing from Matlab

You are working with your math interpreter, in this example MatLab®, and for some reasons you want to plot some data with MattPlot.

A sine wave

An example on how plot a pair of arrays exporting them as a text file.
In MatLab® environment write: 
 To get started, select MATLAB Help or Demos from the Help menu.

» t = (0:0.1:10)';
» u = sin(t);
» tu = [t , u];
» save ('C:\matdats.txt','tu','-ascii','-double','-tabs')
The resulting file will be:
 C:\matdats.txt
  0.0000000000000000e+000	  0.0000000000000000e+000
  1.0000000000000001e-001	  9.9833416646828155e-002
  2.0000000000000001e-001	  1.9866933079506122e-001	
  3.0000000000000004e-001	  2.9552020666133960e-001	
  4.0000000000000002e-001	  3.8941834230865052e-001	
  5.0000000000000000e-001	  4.7942553860420301e-001
  ...
Now you can open it with MattPlot specifying to plot second column respect the first, writing: 
C:\> MattPlot +"#header <t><u(t)>" "C:\matdats.txt"
The chosen columns names u and t are arbitrary.

A couple of tracks

In MatLab® there is a more general way to write files, using file handle and the function fprintf. In the following example we write from MatLab® a ready to use file that can be correctly interpreted by MattPlot without additional lines.
In MatLab® environment write: 
 To get started, select MATLAB Help or Demos from the Help menu.

» N = 1000; % size of tracks
» t = (0:1/N:1-1/N)'; % an independent variable
» y1 = wgn(N,1,0); % a white noise
» y2 = 2*t + 1; % a line
» T = [t,y1,y2]; % the whole data table

» fid = fopen('C:\matdats2.txt', 'wt');
» fprintf(fid, '%s\n', '#header <t><y1(t)><y2(t)>');
» fprintf(fid, '%G\t%G\t%G\n', T');
» fclose(fid);
The resulting file will be:
 C:\matdats2.txt
#header <t><y1(t)><y2(t)>
0	-0.0254305	4
0.001	1.2765	4.002
0.002	1.33929	4.004
0.003	-0.0611252	4.006
0.004	2.33398	4.008
  ...
This file can be opened without adding further informations: 
C:\> MattPlot "C:\matdats2.txt"
