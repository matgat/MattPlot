Select mode

During  SelectMode you can:
  • Change current selected Frame and Track
  • Edit Frames, Tracks and other objects  Properties and style
  • Modify plot layout, namely change Frames position, and move Tracks between Frames.
These operations have generally a change selection effect.

Editing object properties

To access object properties form:
  • Double click on object
  • Right click to access context menu
  • Select object and then trigger  Properties action (for example using the relative main menu entry)

Editing Layout

To drag an object simply left click on it, then move mouse holding left button down. Draggable objects are Frames, Tracks, legends, markers (see  AddMarker), text labels (see  AddText).

With this operation you can:

  • Change current selected Frame or Track
  • Move frames to a new position
  • Swap or merge frames
  • Delete frames releasing them out of plot
  • Move tracks to other frames
  • Delete tracks releasing them out of plot
  • Change legends position

Mouse effects

 SelectMode supports the following mouse operations:
Mouse operation Target Effect
Left click Frame, Track or Legend Object selection; tracks can be selected also on legend
Left double click foil object Edit object properties
Right click Frame or Track Object contextual menu
Wheel Frame Cycle current Track
Left button drag Frame, Track, legend, marker, text label Object dragging
