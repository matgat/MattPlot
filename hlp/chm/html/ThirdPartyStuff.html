Third party stuff

Apart from operating system APIs and IDE libraries, this program uses (or may try to use) the following external libraries:
  • SynEdit for highlighted text editor
  • Anders Melander - Finn Tolderlund TGIFImage for saving in .gif format
  • Planning to add (Not found yet) an XML parser
  • hhctrl.ocx: Microsoft® HtmlHelp library HtmlHelp library
  • Usable compression-decompression dynamic libraries
    • zlibwapi.dll: the well known zlib, by Jean-loup Gailly and Mark Adler, used to handle zipped archives (*.zip) and gzipped file streams (*.gz).
    • Planning to add support for other compressed formats
      • bzip2
      • LZMA
Other contained stuff:
  • Images: I plundered some excellent icons from Linux windows systems (in particular KDE crystalSVG)
