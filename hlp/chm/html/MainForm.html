The main form

The main program window can be divided in the following parts:
  • Main menu: contains all user actions
  • Toolbar: customizable buttons to trigger most used actions
  • Plot area: the Paintbox where frames are rendered
  • Statusbar: contains status informations like current point, action hints, last log entry

Main menu

Placed under the program caption there is the main menu. It ha a fixed structure and contains all actions available to user; it can be disabled changing the EnableMainMenu option entry in the option file.

Toolbar

Placed under the main menu there is the toolbar, a region containing buttons that fire an action subset supposed to contain the most frequently used ones. The toolbar can be disabled with EnableToolbar option and customized editing the Toolbar option entry in the option file.

Plot area

The inner body of the main form is devoted to Tracks rendering. One or more frames are arranged according to a layout.

You can interact with plot area in a number of ways; see Viewing section for detailed informations.

Statusbar

Located at the bottom of the main form, it displays some infos. Clicking on statusbar has the same effect as  ShowLog action.

Keyboard shortcuts

The most useful actions are associated to keyboard shortcuts, customizable by editing the option file. Shortcuts are very important for actions like  Abort and  Snapshot.
