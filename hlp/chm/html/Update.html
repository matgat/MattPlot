Update

To update the program:
  • Close program if running
  • If the option DeleteTemp is set to false delete manually the program temporary directory to assure the creation of the newest help file
  • Overwrite executable with the new one
  • Launch program
  • Activate  StoreOptions action to update ini file
  • If you have a customized MattPlot.ini, give a look to Log to detect any problems. Infact, in case I had the bad idea to rename some options, you will have to move manually your customizations in the new options structure
