Main actions

An action is an implementation-useful concept indicating a generic service provided by the program, like save or quit. Users can request an action interacting with program interface (menu,toolbar,keyboard shortcuts). Note that we will distinguish between actions provided by main program interface and actions provided by the plot subsystem. The following table collects the services provided by the main interface. Most actions require an argument: a target Frame or Track. To view actual shortcuts associated to plot actions look in your MattPlot.ini file. The following table collects all user available actions.
Name Description Note
File
Open Open a plot or data file You can also drag the files into the form
Save Save current plot as MattPlot file Current data and plot layout will be saved
Export Export current plot to a given format This is generally a lossy, irreversible transformation, see Exporting section
ExportFrameToText Export current Frame tracks to a text file This is a shortcut of the more general ExportFrame
ExportFrameToMatlab Export data to MatLab® workspace via m-file This is a shortcut of the more general ExportFrame
Print Print current plot bitmap See printing section
EditOptions Edit the ini file containing the non volatile application settings Avoid to put this action in the toolbar, since it calls ReadOptions
ReadOptions Parse the ini file Useful when you change the ini file with an external editor; it rebuilds toolbar, so cannot be launched from there
StoreOptions Save current options in the ini file Useful to recreate a default MattPlot.ini when missing
SetDefaults Reset options to default values Has the same effect of restarting program without MattPlot.ini
SetStylePlain A style example; this style is good for printing Use  StoreOptions and see MattPlot.ini to view these customizations
SetStylePSpice A style example Use  StoreOptions and see MattPlot.ini to view these customizations
Exit Quit program
Plot
SelectMode Switch to select mode In this mode you can modify plot layout by dragging, adding or deleting frames and tracks
EditMode Switch to edit mode In this mode you can modify tracks; currently it includes only a crop tool
PanMode Switch to pan mode In this mode you can modify Frame mapped space offset
ZoomMode Switch to zoom mode In this mode you can modify Frame mapped space scale
MeasureMode Switch to measure mode In this mode you can perform simple measures on tracks
PenMode Switch to pen mode In this mode you can scribble on plot canvas; those marks will be cleaned at next refresh, however they remains on prints and snapshoots
LockX Synchronize frames abscissa scale Operations on current Frame are reflected on the others, useful when comparing more frames
LockY Synchronize frames ordinate scale Operations on current Frame are reflected on the others, useful when comparing more frames
ClearAll Remove all frames and Track, emptying the plot This action cannot be undone
Snapshot Take a plot bitmap snapshot and save in current directory with a default name This overwrites last snapshot See also SnapshotAction option
Refresh Force a repaint of plot canvas Useful when you want to remove volatile scribbles
MeasureAbscissa Enable instant abscissa measure Affects only measure mode
MeasureOrdinate Enable instant ordinate measure Affects only measure mode
MeasureSlope Enable instant slope measure Affects only measure mode
ViewMeasureMarkers Display measure markers See also measure mode and MeasureMarkers option
Properties Edit generic object properties Applies on last clicked object
Remove Remove the current selected object Applies on last clicked object
StayOnTop Keep the program main form always visible Force main form to stay on top (z order) when focusing other forms
ExecMacro Execute the macro defined in Macro option entry Not yet implemented
Frame
AddFrame Create a new empty Frame
RemoveFrame Delete a frame Applies on current Frame; same effect of dragging Frame outside the form
FrameProperties Edit frame properties Applies on current Frame
ClearFrame Remove all Track in current Frame This action cannot be undone
AddMarker Insert a vertical/horizontal line at a given value Applies on current Frame
AddText Insert a text label at given coordinates Single line text (no word wrap), applies on current Frame
GotoOrigin Force space offset to view frame axes origin Applies on current Frame
FitToAxes Change Frame scale to fit all contained Tracks and axes Applies on current Frame
FitToTracks Change Frame scale to fit all contained Tracks Applies on current Frame
FitToTrack Change Frame scale to fit current Track Applies on current Frame
ZoomIn Change scale to map less space (zoom in) This default zoom is introduced to follow some users practice; it would be better use the quick toggle to zoom mode
ZoomOut Change scale to map more space (zoom out) Same as previous
ToggleScaleX Use linear or logarithmic scale for abscissa axis x=KX+X0 / x=KLogX+X1
ToggleScaleY Use linear or logarithmic scale for ordinate axis y=KY+Y0 / y=KLogY+Y1
LegendToggle Show/hide legend Applies on current Frame; you can also drag legend
GridToggle Show/hide grid Applies on current Frame
GridMore Thicken (shrink) grid lines Applies on current Frame
GridLess Thin (expand) grid lines Applies on current Frame
Track
AddTrack Create a new Track in current Frame
RemoveTrack Delete current Track Same effect of dragging Track outside the form
TrackProperties Edit Track properties Applies on current Track
MarkPoints Plot Track marking its points See also TrackPointSize option
ShowCoords Show Track points coordinates Coordinates are shown when point distance is greater than the value set in TrackCoordsDist option
StrokeLines Plot Track drawing linear interpolation lines Applies on current Track
FillArea Plot Track filling subtended area Applies on current Track
DrawHistograms Plot Track painting a rectangle for each point Applies on current Track
TrackData Edit Track data manually See here for details
TrackTransform Apply a transformation on current Track For example cropping or filtering; see Transforming section
TrackStatistics Display current Track quantities Relative to overall domain, applies on current Track
Tools
ShowTree Show plot tree of contained frames and tracks Drag and drop should be enabled; remember that you can also modify plot layout switching to Select mode and drag frames and tracks with mouse
ShowPane Show/hide a control that displays measured quantities relative to selected Track Use it with Measure mode. See also AutoPane option.
ShowLog Toggle log window Log is useful to know what is happening
Compute Elaboration tool: perform operations between tracks Not yet available
TextEditor Run an instance of embedded text editor Take advantage of syntax highlighting to edit data or ini files
Abort Stop current long-time action Typically when importing or computing a large amount of data. Advice: remember its keyboard shortcut!
Help
Update Manually check for updates This simply opens current MattPlot web site address in your default browser
Demo Add some frames and tracks to demonstrate program capabilities Use this for training purposes
ExperimentalStuff Another demo with ugly stuff Not recommended features
HelpContents Opens this file Shortcut is fixed to F1. Uses HtmlHelp library (see here)
About Display program informations Also contains a simplified performance indicator
