How to...

Placing markers

Try  AddMarker action; you can then drag the marker during select mode.

Analyze spectrum

To understand how it works, try to follow these steps:
  • Open MattPlot (see here)
  • Start  Demo, you'll see a Frame containing a Track named Sum, given by the sum of three harmonics
  • You may want remove unusued Frames switching to  SelectMode and dragging them out of plot area or using  RemoveFrame
  • Start  TrackTransform
  • Select the Track Sum and then the unary operation Spectrum, fft or dft, then press Do
  • A new Frame appeared containing the spectrum of Sum
  • Close the  TrackTransform window and inspect the spectrum, which should contain the proper harmonic components

Paste plot image in documents

You can easily paste plots as vectorial image in your WYSIWYG editor:
  • Check SnapshotAction option
  • Do  Snapshot action to copy the plot image in the clipboard
  • Switch to your editor and place cursor in the desired position
  • Paste the file (CTRL+V)

Taking snapshot with volatile text

User can take a snapshot of plot with volatile text (like pen comments and measure lines) triggering the  Snapshot action, after setting properly the SnapshotAction option to generate a bitmap.

Comparing two acquisitions

MattPlot provides functions to ease data comparison. Suppose you want compare two set of data, for example two oscilloscope acquisition of the same quantities done at different conditions, stored in the two files data1.txt and data2.txt. To compare them, first load these data in two different frames; you can do that either by drag and drop, or by  Open action, or by command line typing: 
C:\> MattPlot data1.txt
C:\> MattPlot @data2.txt
The prefix @ in the second call prevents a further instance of MattPlot.

At this point operations on frames like  PanMode or  ZoomMode are spread on all frames.

Manually sampling a track

You can extract some points from a track using mouse following these steps:
  • Open the log with  ShowLog; if you desire, you may dock it on the form right border
  • Zoom your track properly
  • Switch to  MeasureMode and select track
  • It is handy enable  MeasureAbscissa and  MeasureOrdinate
  • Move mouse and double click on desired points
  • When you finished, select and copy the coordinates in log and paste in your text file; you can then import it back in MattPlot, eventually adding an header like #header<X><MySamples(X)>
