Pan mode

During  PanMode you can modify Frame mapped space offset, in other words you can shift plots moving along plane directions. Use  LockX or  LockY actions to involve all frames.

Mouse effects

 PanMode supports the following mouse operations:
Mouse operation Target Effect
Left button drag Frame Frame space dragging
Right button drag Frame Continuous Pan feature
Left double click or middle click Frame Reset position returning to axes origin
Wheel Frame Horizontal Pan (default increment)
Wheel + Right button Frame Vertical Pan (default increment)
Left-Right click Frame Selects frame
