Launching the executable

You can run MattPlot in two ways:
  • Double clicking on executable file
  • launch it from the command line
In the following example we are supposing to have the executable file named MattPlot.exe placed in the shell current directory or in the executables search path list: 
C:\> MattPlot
If you don't like its name, you may set an environmental variable: 
C:\> set mp="C:\Program files\My uts\MattPlot.exe"
C:\> %mp%
