Printing plot

You can print plots in two ways:
  •  Export plot as image and then print it with an external program; this can be useful to include also measure lines and other volatile stuff.
  • Use embedded  Print function.
MattPlot printing function can work in two ways:
  • Print screen bitmap - takes what you see on the screen and stretches it in paper area
    • feature: preserve plot aspect ratio, so paper area may not be optimally filled
    • advantage: should includes also temporary scribbles of pen mode
    • drawback: low image resolution
  • Redraw on print paper - redraws plot in the printed page canvas
    • feature: changes plot aspect ratio to fill paper area
    • advantage: best resolution
    • drawback: temporary objects like scribbles are lost
In both cases you can set the paper margin percentage editing the PrintMargin option.
