Pen mode

During  PenMode you can scribble on plot canvas; this tool is meant to be used in combination with  Snapshot action and works only with bitmap output. You have two customizable tools: pen and highlighter; hold down respectively left or right mouse button to use them. Customize them modifying the option entries: PencilSize, PencilColor, HighlighterSize, HighlighterColor.
Warning: the scribbles drawn in this way are ephemeral, and will be cleaned at next repaint; however they are freezed when Canvas is transformed in a bitmap, when using  Export or  Snapshot action, in this case setting properly the SnapshotAction option.

Mouse effects

 PenMode supports the following mouse operations:
Mouse operation Target Effect
Left button drag Frame Draw with pencil
Right button drag Frame Draw with highlighter
Middle click Frame Refresh erasing all scribbles
Wheel Frame Default horizontal Pan
Wheel + Right button Frame Default vertical Pan
