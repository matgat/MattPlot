Exporting to bitmap

With Bitmap I mean a two dimensional array of colors, either compressed (typically with lossy algorithm) or not.

Screen bitmap

You can capture and save the plot bitmap rendered on screen using  Export or  Snapshot actions, setting properly the SnapshotAction option.

High resolution bitmap

You can obtain a bitmap having an higher resolution than screen in two ways:
  • Using the print function in combination with a virtual printer like pdfcreator
  • Exporting to metafile and converting it to bitmap with an external tool
