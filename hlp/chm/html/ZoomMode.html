Zoom mode

During  ZoomMode you can modify Frame mapped space scale. Use  LockX or  LockY actions to involve all frames. You can zoom in two ways:
  • Zoom drawing a zoom rectangle (see the SuddenZoom option)
  • Default zoom respect current cursor point (see ZoomPerc option)
Some customizations pertaining zoom mode are described in ZoomMode options.

Mouse effects

 ZoomMode supports the following mouse operations:
Mouse operation Target Effect
Left button drag Frame Build a zoom rectangle
Left click Frame Selects frame; if inside zoom rectangle perform zoom in, if outside delete it; if no rectangle perform default zoom in: hold right button to zoom out abscissa only see also the SuddenZoom option
Right click Frame Default zoom out (respect cursor pos); hold left button to zoom out abscissa only
Left double click Frame Reset scale to fit all tracks
Middle click Frame Default zoom in (respect cursor pos)
Wheel Frame Default zoom in-out; hold left or right mouse buttons to zoom only ordinate or abscissa (respect cursor pos)

Pixel size in physical space

Zoom operation lead to a change of proportion between screen representation and represented physical space, that is the pixel size in physical space units.

This quantity is quite important because affects all measures depending on mouse position (current point, interval average, etc...). It is worth to observe that to increase the accuracy of these measures there are two ways:
  • Zoom in as possible the region of interest
  • Increase your media pixel resolution
The output numbers reflect this granularity, however in this mode you can get exact values of pixel size in the statusbar (in place of current point) holding left click on the frame of interest.
