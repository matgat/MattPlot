What is

MattPlot is a simple program whose primary aim is to plot data, more precisely to render an interactive 2-d graphical representation of a set of arrays of numbers. It comes to facilitate a typical task of engineering: view and analyze of a set of collected values.
This program is meant to:
  • Plot data contained in a text file
  • Allow user to inspect data and perform simple measurements
  • Perform some simple postprocessing and data elaboration
  • Export plot to text and image (ex, to include in WYSIWYG editors)

Why another data plotting tool?

For the usual reason: haven't found what I wanted. I missed a simple, fast and complete tool to visualize the measures collected in a servo drive, so I decided to write from scratch a new one for me.
