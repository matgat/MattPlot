Program related files

A brief description of program related files.
File name Description Note
MattPlot.exe Windows® 32 bit executable The program itself (no other files neeeded)
DefaultDir The working directory Set in DefaultDir option entry
TempDir The temporary directory (\~MattPlot\) Set in TempDir option entry
MattPlot.chm Microsoft™ compiled HTML help Generated in TempDir at request. Requires a HtmlHelp viewer (hh.exe, generally present in all Windows® systems)
MattPlot.ini Program options file Fixed name and position (same exe directory); initially not present, create with  StoreOptions
*.mpl MattPlot`s text data file Plot data in text format; this is a fake extension that provide the possibility to filter files and associate MattPlot with them
*.mpx MattPlot`s XML plot description format Plot complete description in XML format
*.log MattPlot`s crash log files A text file containing the program log lines, autosaved in case of critical errors
