The option file

At each execution MattPlot searches in its directory the options ini file having the fixed name MattPlot.ini; this file allows user to customize the program. When the option file is missing, the program will use default values (the log warns about this occurrence). To read again the ini file use  ReadOptions action, or drag it in the program form. Note that some settings may apply only at next program run.

The ini file format

Initialization file, or ini file, is a text file format introduced by Microsoft™ to store configuration data in a flexible way. This format is constituted by three elements: sections, parameters and comments; each parameter definition comprises name and value.
 Notepad
[section1]

; a comment
name1=value1

[section2]

name2=value2
name3=value3
MattPlot has an its own ini parser implementation that may differ from standard behaviour; depending on types, sometimes value strings are trimmed (removing of leading-trailing spaces and control characters). In such cases you must use quoted strings and escape sequences to specify special strings like:
 an entry with ctrl chars
somedelims="\ \t\n\r"
