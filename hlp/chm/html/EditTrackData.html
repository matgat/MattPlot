Edit data manually

Manual edit

Although its use is improbable, this tool is introduced for completeness. The  TrackData action opens an editor containing data columns: the first is the independent variable, the remaining is the dependent one.
This conversion mechanism is very simple and has no check on typing errors; if you aren't sure of what you're doing, better export to text, edit file, and then import back.

Using EditMode

The  EditMode provides some tools like crop the track abscissa interval (real tracks only).
