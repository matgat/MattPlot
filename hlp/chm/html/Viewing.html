Viewing data

Analyzing a Track graphically involve the capability to change physical space representation on screen (the scale) and then retrieving from represented points some useful quantities. Here are the tools MattPlot provides.

  • Viewed data

    on the difference between rendered and measured data

  • Select mode

    change selection and plot layout

  • Edit mode

    modifying data and crop tool

  • Pan mode

    drag domain offset

  • Zoom mode

    magnify or dwindle frame view

  • Measure mode

    retrieve frame points and track quantities

  • Pen mode

    scribble and highlight plot
