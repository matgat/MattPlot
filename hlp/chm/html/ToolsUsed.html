Tools used

During development, the following excellent products were used:
  •  Borland C++ Builder® 5 - the C++ IDE
  • Subversion and tortoiseSVN - versioning system
  • winmerge - the best program to compare text files
  • ConTEXT and then PSPad: nice text editors, mainly used for XML files
  • Saxon - XSL interpreter
  • ImageMagick - to get the .gifs in this help
  • UPX - the legendary Ultimate Packer for eXecutables
  • pdfcreator - .pdf printer driver
  • 7z - good archiver
