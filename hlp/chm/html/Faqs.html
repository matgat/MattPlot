1) Program arguments
1.1) How can I prevent the creation of further program instances when I open other files from command line?
Prefix program arguments string with the special character @. See related section.
1.2) How can I load two files in the same plot frame?
By mouse: drag the first file in plot area, then the second in the same frame of first file
By command line: specify the option -f (see here) as in the following example: 
C:\> MattPlot -f file1.txt file2.txt
2) Importing data
2.1) Can I rescale (multiplication by a constant) my input data?
You can do it at parsing time using text data file format, in particular the parser's Header directive, for example:
 an acquisition
#header <y [0.5]>
// following data will be
// multiplied by 0.5
1
2
3
A more complex example:
 an acquisition
#header <t[1E-3s]><V(t)[10V]><i(t)[0.1A-2]>
0.0   142.21   8.1
0.1   134.65   8.7
0.2   117.32   9.1
In this example we have three measures taken every 100us of a voltage and a current, the first expressed in tens of volt, the second in tenths of ampère; to the current is also applied an offset of -2 after rescaling (so this means -2A).

Remember that you can rescale tracks or perform other elaborations using the  TrackTransform tool (see Transforming section).

2.2) Can MattPlot open or save as compressed files?
MattPlot can handle some compression formats, see here.
3) Viewing plots
3.1) Can I temporary toggle to pan or zoom mode while measuring track?
Yes, use the PanToggle and ZoomToggle plot actions whose key shortcuts are defined in options file.
3.2) I zoomed too much, how can I reset plot view?
Use one of the following actions:  FitToAxes,  FitToTracks,  FitToTrack; set the shortcuts using ShortCuts option entries. Note that also some plot keys like cycle may perform fit operations; furthermore remember that during zoom mode you can reset scale with a mouse middle click.
3.3) I want to edit the properties of a track, but selection with mouse is difficult. Is there another way?
Select the track using plot keys and then fire the  TrackProperties action.
3.4) Can I get a plot image comprising the current measure lines?
Yes, see this tip.
3.5) How can I record some point coordinates of a displayed track?
In measure mode you can record track points in the log; see related section.
4) Customizing
4.1) There is an action that I use often; is there a way to call it quickly avoiding the main menu?
Check if there is an available keyboard shortcut set in ShortCuts section of the option file, refer to string-key table to customize them.
4.2) No shortcut available for my beloved action, Am I condemned to access it through the slow main menu for the rest of my life?
You can add a button on the toolbar. Edit the action names sequence in Toolbar option. Get action names from the actions table or navigating main menu and looking at the statusbar hints.
